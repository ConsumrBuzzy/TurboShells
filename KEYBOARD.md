# Turbo Shells - Keyboard Shortcuts

## Global Navigation
- **M** - Return to Main Menu (from any screen)

## Main Menu (Stable)
- **R** - Start Race
- **S** - Go to Shop
- **B** - Go to Breeding

## Turtle Management (Main Menu)
- **Q/W/E** - Train turtle in slot 1/2/3
- **Z/X/C** - Rest turtle in slot 1/2/3
- **4/5/6** - Retire turtle in slot 1/2/3

## Shop
- **R** - Refresh shop inventory
- **1/2/3** - Buy turtle in slot 1/2/3

## Race Screen
- **1** - Set race speed to 1x
- **2** - Set race speed to 2x  
- **3** - Set race speed to 4x

## Race Results Screen
- **M** - Return to Main Menu

## Breeding Screen
- **1/2/3** - Select parent turtle 1/2/3
- **Enter** - Breed selected parents

## Notes
- Mouse controls are also available for all buttons and interactions
- Keyboard shortcuts are provided for quick access to common actions
