# TurboShells Screenshot Capture Guide

## Overview
Capture 5 key screens for portfolio documentation. Save all files to `assets/screenshots/`.

---

## 1. Race Simulation (GIF) ⭐ HERO IMAGE
**File:** `race_simulation.gif`  
**Duration:** 5-10 seconds  
**Resolution:** Full window or 1280x720 crop

**What to capture:**
- Race in progress with 2-3 turtles
- Turtles moving through different terrain (water/rocks visible)
- Energy bars depleting and recovering
- Position changes (lead changes are ideal)

**Tip:** Start recording just as race begins, capture ~5-10 seconds of action.

---

## 2. Main Menu / Roster View (PNG)
**File:** `roster_view.png`  
**Resolution:** Full window

**What to capture:**
- Active roster showing 2-3 turtles
- Turtle cards with **visible genetics** (shell colors/patterns)
- Stats visible (Speed, Energy, etc.)
- Money/balance visible in corner

**Highlight:** The procedurally generated turtle sprites - no two look alike!

---

## 3. Shop View (PNG)
**File:** `shop_view.png`  
**Resolution:** Full window

**What to capture:**
- Shop inventory with available turtles
- Price tags visible
- Player's current balance
- At least one turtle with interesting genetics

---

## 4. Turtle Profile / Detail View (PNG)
**File:** `turtle_profile.png`  
**Resolution:** Full window

**What to capture:**
- Single turtle's detailed view
- Full stats breakdown
- Genetic traits (shell pattern, colors)
- Race history if visible

**Highlight:** The 19-trait genetic system controlling appearance.

---

## 5. Breeding Screen (PNG or GIF)
**File:** `breeding_view.png` or `breeding.gif`  
**Resolution:** Full window

**For PNG:**
- Two parent turtles selected
- Child preview showing inherited traits

**For GIF (optional, 3-5 sec):**
- Selecting parents → breeding → seeing offspring

**Highlight:** Mendelian inheritance visually demonstrated.

---

## Technical Notes

| Format | Use For | Tool Suggestion |
|--------|---------|-----------------|
| PNG | Static screens | Win+Shift+S, ShareX |
| GIF | Race action, breeding | ShareX, ScreenToGif, LICEcap |

**Recommended dimensions:** 
- Hero GIF: 800-1200px wide
- Screenshots: Full window or 1280x720

**File naming:** All lowercase with underscores
